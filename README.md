## Project Overview

## Key Features
* **AOG Risk Heat-Map**: Visualizes regional bottlenecks with a focus on Central America and Western Europe.

* **Aerospace Taxonomy**: Transformed raw supply chain data into industry-specific categories like **Sensors**, **Hydraulic Systems**, and **Avionics**.
